Method  
- Input Layer: Bias + 1 binary input
- Output Layer: 2 outputs with liklihoods for `1` and `0`.
- Activation function: Sigmoid
  - S(x) = 1 / (1+e^-x)
- Using Stochastic Gradient Descent for Backpropagation
