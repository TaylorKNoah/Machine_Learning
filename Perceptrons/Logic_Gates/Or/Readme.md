Method  
- Input Layer: Bias + 2 binary input
- Hidden Layer: None
- Output Layer: 1 output.
- Activation function: Sigmoid
  - `a = S(x) = 1 / (1+e^-x)`
- Threshold: `0 < a < 0.5`
- Using Stochastic Gradient Descent for Backpropagation
  - Update weigts after each training example
